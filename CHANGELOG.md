# 更新日志

## 1.0.1 - 2026-04-28

本版本在 1.0 基础上补齐真实使用场景下的前端体验、部署能力和工作流细节。

### 新增

- 接入后端真实接口，支持登录、资料采集、资料流转、主题和标签管理。
- 新增收件箱、资料库、搜索、主题设置、统计、个人中心和灵感助手入口。
- 支持未读/已读状态、多端同步、未读筛选和刷新后未读置顶。
- 新增暗夜模式、通透风格、二次元风格和主题色自定义。
- 新增沐沐看板娘，支持拖拽、缩放、隐藏唤回和说话动画。
- 增加 Docker 部署文件和中文部署说明。

### 优化

- 优化桌面端三栏工作台布局，资料列表和详情区独立滚动。
- 优化移动端导航、筛选、采集弹窗、详情页滑动和图片预览体验。
- 资料库和搜索页支持分页加载、状态筛选、标签分组筛选和评分排序。
- 资料列表展示封面、标签、评分、评语摘要和更新时间。
- 二次元风格下支持随机壁纸、封面兜底和更透明的详情背景。

### 修复

- 修复登录后空白页、CORS 代理路径、状态流转提示不明显等问题。
- 修复收录、评价、失效、删除标签等操作中的前后端状态不一致问题。
- 修复图片资料详情页移动端无法滚动、横向溢出和原图预览问题。
- 修复默认封面在非二次元风格下丢失的问题。
